# Linkedin Scraper

A simple Linkedin Scraper that uses Google to search for Linkedin Profiles

## Requirements

* python2.7+
* pip
* requests

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python linkedin_scraper.py --keyword "nasa" --limit 500
```
